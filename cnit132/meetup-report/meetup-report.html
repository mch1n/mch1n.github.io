Meetup Report

I participated in the online meetup hosted by PunchCode based in Las Vegas. I attended the online version since SF still has shelter-in-place due to COVID-19.

Details of the meetup:

Tuesday, March 24, 2020

Free ONLINE Crash Course - Intro to Front End Engineering w/ JavaScript

Hosted by: PunchCode

Meetup event link: https://www.meetup.com/PunchCode/events/269555866/

PunchCode is a coding bootcamp provided by Tech Impact. Tech Impact is a nonprofit whose mission is to empower communities and nonprofits to use technology to better serve our world. PunchCode holds these meetups once a month to get people to check out their coding bootcamp and to get people more interested in learning to become software engineers.

The online meetup was an introductory crash course in Javascript. The first half was an brief intro to the various types of JS variables, comparison operators, conditional statements, and a brief overview of functions: Primitve types (compared by value - immutable, so no way to change after it's created)

  • Number
  • String
  • Boolean
  • Null
  • Undefined

Non-primitve types (compared by reference - mutable, so value can change after it's created)

  • Array
  • Objects - functions

Comparison operators (some examples)

  • =, ==, ===, >, <, !=, +, -, *, /, &&, ||

Conditional statements

  • if-else statements
  • for loops
  • while loops

There was also a demo of a JS number game where you guess a number and the game will give you hints if you're close to the number or not.

Code was hosted on codepen and he went through the various functions of the game. From defining variables, using Math.random(), and Submit event listener and then a function to capture the submit event. There was also conditional statements to check if the number was correct or keep guessing a lower or higher number.

Overall, it was a very informative session, and it was a good overview intro to JS.
